## 🎯 Objective

To analyze the relationship between **trader behavior** and **market sentiment (Fear vs Greed)** and uncover how emotional market conditions influence trading patterns.

---

## 📚 Project Overview

This project uses trading data along with sentiment indicators to understand behavioral trends in financial markets. The workflow includes:

- 📊 Exploratory Data Analysis (EDA)  
- 📈 Sentiment-driven behavioral analysis  
- 📉 Visualization of trading trends  
- 🧠 Insight extraction for decision-making  

The entire analysis was performed using **Google Colab**, a cloud-based notebook platform widely used for data science and machine learning workflows.

## 🔍 Key Analysis Performed

### 📌 Notebook 1 (EDA)

- Data loading and preprocessing  
- Handling missing values  
- Distribution analysis of trading metrics  
- Identification of key variables affecting trading  
- Visualization of data trends  

### 📌 Notebook 2 (Analysis)

- Mapping sentiment (Fear vs Greed) to trading activity  
- Comparative analysis across sentiment phases  
- Correlation analysis between sentiment index and trading metrics  
- Behavioral trend identification  

---

## 📊 Key Insights

- 📈 **Greed phases** show increased trading activity and higher risk-taking  
- 📉 **Fear phases** indicate reduced trading volume and cautious behavior  
- 🔁 Traders tend to **overtrade during extreme sentiment conditions**  
- 📊 Sentiment index shows **clear correlation with trading intensity**
